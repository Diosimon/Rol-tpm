📖 Instrucciones para el Rol de Reporte TPM
  1. Identifica el “Ritmo” correspondiente al día. La columna Ritmo indica el orden principal de quién debe realizar el Reporte TPM.
  2. El rol sigue un patrón rotativo continuo: 1 → 2 → 3 → 4 → 5 → 1…
  3. Si la persona asignada (Ritmo) no asiste:
    • El Suple 1 toma la responsabilidad del reporte.
    • Si también falta Suple 1, entra Suple 2; si tampoco está, entra Suple 3; si tampoco está, entra Suple 4 como último recurso.
  4. El ciclo no se detiene ni se “recuperan” turnos.
💡 Ejemplo

Si hoy le toca a Christian (Ritmo 1) y no vino, el Suple 1 (Vicente) realiza el reporte TPM.

Al día siguiente el ritmo avanza normalmente al siguiente número (Ritmo 2), por lo que el responsable original del día siguiente sigue siendo Vicente.

Es decir: el sistema avanza un ritmo por día, independientemente de quién terminó haciendo el reporte ese día.
